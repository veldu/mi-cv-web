# Portfolio Web Interactivo

Este es el código fuente de mi portfolio personal. El proyecto está enfocado en crear una experiencia de usuario fluida y moderna, usando animaciones avanzadas y "scrollytelling" para presentar mi experiencia profesional.

# Tecnologías Utilizadas

Vite: Entorno de desarrollo y empaquetado de alta velocidad.

TypeScript: Para un código más robusto y mantenible.

GSAP (GreenSock): La principal librería de animación, usada para:

HTML5 y CSS3: Estructura y estilos modernos.

# Cómo ejecutarlo localmente

Asegúrate de tener Node.js instalado.

Abre una terminal en la carpeta del proyecto e instala las dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```

¡Y listo! El proyecto estará corriendo en http://localhost:5173 (o el puerto que indique Vite).
